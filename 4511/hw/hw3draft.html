Introduction

In this homework assignment, you will write a basic scheduling agent for an air freight terminal. This is a simplification1 of a real-world problem: creating a schedule for the loading and unloading of cargo.

1 An extreme simplification.

The Problem

An air cargo terminal receives aircraft loaded with freight, unloads the aircraft, sorts the cargo, and transfers the cargo to trucks for distribution.

Arrivals

Aircraft arrive on a schedule. Each arriving aircraft has:

  • An arrival time
  • A size: S, M, L or XL
    • Small aircraft carry up to 4 pallets
    • Medium aircraft carry up to 7 pallets
    • Large aircraft carry up to 10 pallets
    • EXtra Large aircraft carry up to 15 pallets
  • A cargo manifest:
    • The manifest indicates how many pallets of each type are carried on the aircraft
    • There are four types of pallets, A, B, C, D

Scheduling: Aircraft must be scheduled to unload at hangars at specific times.

Hangars

Aircraft are unloaded at hangars. Each terminal hangar can handle one aircraft at a time.

  • There are two sizes of hangar, L and XL
    • L hangars can accommodate S, M, and L aircraft
    • XL hangars can accommodate all sizes of aircraft

After pallets are unloaded, they are stored at the hangar until they are subsequently loaded onto a cargo truck

Forklifts

Aircraft are unloaded by forklifts. Each forklift can unload one pallet in 20 minutes.

  • Any number of forklifts can work to unload a plane at any time
  • One pallet can only be unloaded by one forklift
  • To illustrate:
    • If a plane is loaded with two pallets, one forklift will unload the plane in 40 minutes
    • If a plane is loaded with two pallets, two forklifts will unload the plane in 20 minutes
    • If a plane is loaded with two pallets, three forklifts will unload the plane in 20 minutes
      • Each of the two pallets is already being unloaded, so the third forklift has nothing to do
  • Assume it takes no time for a forklift to move from one hangar to another

Scheduling:

  • Forklifts must be scheduled to work at hangars at specific times.
  • Forklifts must be scheduled to unload an aircraft or load a truck (see below)

Trucks

Each cargo truck can receive one pallet. Cargo trucks are loaded by forklifts, similar to how aircraft are unloaded by forklifts. Each forklift can load one pallet in 5 minutes:

  • A cargo truck can be loaded by one forklift in 5 minutes
  • A cargo truck can be loaded by two forklifts in 5 minutes
    • The pallet is already being loaded, so the second forklift has nothing to do
  • Each hangar can accommodate one truck at any given time
  • Assume the truck leaves as soon as it is loaded

Scheduling:

  • Cargo trucks must be scheduled to be loaded at hangars at specific times.

Scheduling

  • Each aircraft must be scheduled to start unloading at a specific hangar at a specific time
    • The time must be after the plane arrives
    • The hangar must be able to accommodate the aircraft’s size
    • Assume that the aircraft will leave the hangar when it is finished unloading
  • Forklifts must be assigned to a specific hangar
    • Each forklift must be assigned to load or unload a specific pallet type at a specific hangar at a specific time
    • A forklift can only unload if there is a plane waiting to be unloaded
    • A forklift can only load if there is a truck waiting to be loaded and a pallet to load

Time is always expressed as minutes on a 24h clock, i.e. 900 is one minute after 859 and 1300 is one minute after 1259. Events should be scheduled to the minute (not more granularly).

The scheduling problem is solved if an assignment of aircraft, forklifts, and trucks results in all cargo being unloaded from aircraft and loaded onto trucks by the end of the day.

Data Format - Problem

The meta.json file indicates what hangars are available and what forklifts are available:

meta.json
{
  "Hangars": {"North Hangar": "L", "New Hangar": "XL"},
  "Forklifts": ["F01", "F02"]
}
  • Hangars will be named via an arbitrary unique string.

The aircraft.json file indicates the arrival schedule of aircraft:

aircraft.json
{
  "D213": {
    "Size": "L", "Time": 745, "Cargo": {"A": 0, "B": 1, "C": 2, "D": 0}
    },
  "R Special": {
    "Size": "S", "Time": 820, "Cargo": {"A": 0, "B": 0, "C": 0, "D": 1}
    },
}
  • Aircraft will be named via an arbitrary unique string.

The trucks.json file indicates the arrival schedule of aircraft:

trucks.json
{
  "JLX-9828": 645,
  "PK2-341": 715,
  "ADR-005": 720,
  "TX-241-Y": 755,
}
  • Trucks will be named via an arbitrary unique string.

How To Solve It

  • First, frame the problem as an assignment of values to variables:
    • You will need to determine what the variables are
    • You will need to determine what the domains of each variable are
    • You will need to express the constraints
  • You could solve the problem directly using backtracking search
    • Implementing AC3 may speed up your search
  • You could frame the problem as a SAT problem and use a SAT solver such as pycosat
